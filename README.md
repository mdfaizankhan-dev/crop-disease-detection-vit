# Crop Disease Detection using Vision Transformer (ViT)

## Project Overview
This project detects tomato leaf diseases using a Vision Transformer (ViT) deep learning model.

## Classes
- Tomato_Early_Blight
- Tomato_Healthy
- Tomato_Late_Blight

## Technologies Used
- Python
- PyTorch
- timm
- Flask
- HTML/CSS

## Project Structure
├── app.py
├── train.py
├── predict.py
├── requirements.txt
├── templates/
├── static/

The trained model file (crop_disease_model.pth) is not included in the repository because of GitHub file size limitations.
